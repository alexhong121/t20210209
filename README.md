# Markdown 語法
## 這是標題 `#`
--- 
標題總共有1到6 根據 # 數量, 1最大 6最小 

ex:

### 第一種 '###'
#### 第二種 '####'
****************
## 區塊引言 `>`

>this is blockquote
>>有很多變化
>## 引言+標題

--------------------------------
## 清單 `*`

### 示範 `*`
* red 
* black
* green
  
### 示範 '+'
+ red
+ green
+ black

還有 '-' '1.'
1. red
2. green
